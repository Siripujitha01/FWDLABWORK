FRONT END WEB DEVELOPMENT

The HyperText Markup Language, or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.
Cascading Style Sheets is a style sheet language for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript. It describes the look and formatting of a document which is written in a markup language. It is generally used with HTML to change the style of web pages and user interfaces.
JavaScript, often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web,alongside HTML and CSS. JavaScript is a simple and easy-to-learn programming language as compared to other languages such as C++, Ruby, and Python. It is a high-level, interpreted language that can easily be embedded with languages like HTML. It was developed by Netscape Communications Corporation, Mozilla Foundation, and ECMA International.
